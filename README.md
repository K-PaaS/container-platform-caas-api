## 시작하기

K-PaaS에서 제공하는 컨테이너 서비스 제어에 필요한 REST API를 제공합니다.

### 빌드 방법

컨테이너 서비스 API 소스 코드를 활용하여 로컬 환경에서 빌드가 필요한 경우 다음 명령어를 입력합니다.
```
$ gradle build
```

## 개발 환경

개발 환경은 다음과 같습니다.

| Situation                      | Version |

| JDK                            | 8       |
| Gradle                         | 4.4.1   |
| Spring Boot                    | 2.0.4   |
| Apache Http Client             | 4.5.6   |
| Lombok                         | 1.18.2  |
| Gson                           | 2.8.5   |
| Jacoco                         | 0.8.1   |
